## 📌 What Is This Project?

**CyberShield Academy** is a structured, beginner-friendly cybersecurity learning project that takes you from zero knowledge to being able to perform real penetration tests, analyze forensic artifacts, and respond to incidents — in 14 weeks.

This repository contains:
- 📚 **8 learning modules** with theory + practical labs
- 🚩 **9 CTF (Capture The Flag) challenges** with full writeups
- 🔬 **5 hands-on labs** using industry tools
- 📝 **Report templates** used by real security professionals
- 🛠️ **Automation scripts** in Python & Bash
- 📊 **Project tracker** and team task board

## 🚀 Quick Start

### Prerequisites

```bash
# You need:
# - Kali Linux (or Parrot OS) — recommended as a VM
# - Python 3.8+
# - Git

# Clone the repository
git clone https://github.com/Debasish-Nayak-656/CyberShield-Academy.git
cd CyberShield-Academy

# Install Python dependencies
pip install -r requirements.txt

# Run the environment checker
python tools/env_check.py
```

### Setting Up Your Lab Environment

```bash
# Option 1: Use VirtualBox/VMware with Kali Linux
# Download Kali: https://www.kali.org/get-kali/

# Option 2: Use Docker for web labs
cd labs/lab-03-burpsuite
docker-compose up -d

# Option 3: Use TryHackMe/HackTheBox online (free tier available)
# Links in docs/online-labs.md
```
